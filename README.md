Brando
==================================================
[![serverless](http://public.serverless.com/badges/v3.svg)](http://www.serverless.com)

Brando is an API (using lambda) managing the db containing live.tortuga.cloud 's events

### Install serverless
```shell
$ npm i -g serverless
```

### Build
```shell
$ make build
```

### Test
```shell
$ make test
```

### Test, build & deploy Dev
```shell
$ make deploy
```

### For Prod
```shell
$ make deploy-prod
```
